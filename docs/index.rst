Sticky Notes Documentation
===========================

Welcome to the Sticky Notes application documentation. This Django-based web application
allows users to create, view, edit, and delete sticky notes with an optional author assignment.

Features
--------

* Create sticky notes with title and content
* Assign authors to notes
* View all notes in a list
* View individual note details
* Edit existing notes
* Delete notes
